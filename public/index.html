How it works

Reachable areas come from several providers, all proxied through a small Cloudflare Worker so the API keys stay on the server and never appear in this page.

Mapbox gives detailed shapes fast, capped at 60 min. ORS also gives detailed shapes but caps driving at 60 min. HERE gives detailed shapes with no practical time limit. ArcGIS gives detailed drive-time areas (driving only), long range. TomTom gives coarser, traffic-aware shapes, long range. Auto uses fast detailed Mapbox up to 60 min, then HERE/ArcGIS (detailed) and finally TomTom for long drives. The travel-time limit adjusts to the chosen provider.

Tap ⚖ on any location to run every provider at that point and compare how far each one's boundary reaches — areas are overlaid on the map in each provider's colour, with a ranked list by size.

Each location runs at several time bands; only the outer band is used for the overlap, the inner bands are drawn for context. Each pin shows which provider ran it and the outer shape's vertex count (e.g. ORS 312pt) so you can see how detailed it is.
